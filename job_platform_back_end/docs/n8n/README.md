# TALENTO — Unified CV & Job Matching Pipeline (n8n)

`TALENTO-Unified-Pipeline.json` merges the three original workflows into one,
with every integration bug against the Laravel backend fixed. Generated by
`build_unified_workflow.py` (re-run it to regenerate the JSON).

## What's inside (one workflow, three entry points + shared error handling)

- **Lane A — CV Ingestion**: Google Drive trigger → download → Mistral OCR →
  Gemini classification → 4 typed chunks → embed (OpenRouter) → Qdrant
  `talento_cv_embeddings` + `POST /api/v1/resumes/chunks` + `/api/v1/resumes/embeddings`,
  then **auto-runs Lane C** for the new candidate.
- **Lane B — Job Published**: webhook `/job-embed` (from the Laravel listener) →
  embed → Qdrant `talento_job_embeddings` → confirm → search CVs → aggregate →
  `POST /api/n8n/match/done`.
- **Lane C — CV → Job Matching (shared)**: reachable from Lane A *or* the
  `/cv-match` webhook → fetch candidate chunks → weighted-average vector →
  search jobs → `POST /api/n8n/match/done`.
- **Lane E — Unified error handling**: single Error Trigger → `POST /api/n8n/log-error`.

## Fixes applied vs. the original three

1. **Download file** now uses the triggering file id (`{{ $json.candidateId }}`) —
   was hardcoded to `resume2.pdf`, so every run processed the same CV.
2. **Job text** uses Laravel's prebuilt `text_to_embed` (falls back to raw
   fields) — the old node rebuilt from `description`/`requirements`/`responsibilities`
   that Laravel never sends, so jobs were embedded on the **title only**.
3. **CV→Job match callback** now sends `X-N8N-Webhook-Secret` — the old node
   omitted it and got **401** from the secret-guarded `/api/n8n/*` routes.
4. **Unified error endpoint**: all failures → one secret-guarded `/api/n8n/log-error`
   (was split between `/api/v1/n8n/log-error` and `/api/n8n/log-error`).
5. **Orphan fixed**: the CV pipeline now triggers matching in-process after all
   chunks are upserted (`/cv-match` was previously never called by anyone).
6. **One token + one secret** across every Laravel call.
7. Minor: Gemini model id set to `models/gemini-2.0-flash` (was `gemini-3.5-flash`).

## Import & cut-over

1. n8n → Workflows → Import from File → `TALENTO-Unified-Pipeline.json`.
2. Re-link credentials if ids differ (Google Drive, Mistral, Gemini, Bearer Auth).
3. **Deactivate the three old workflows first** — the new one reuses webhook
   paths `/job-embed` and `/cv-match`; two active workflows on the same path collide.
4. Activate the unified workflow.

## Inbound webhook authentication

Both inbound webhooks (`/job-embed`, `/cv-match`) are now gated by an **IF node**
that checks the `X-N8N-Webhook-Secret` header against `talento-secret-2026`:
- valid  → responds 200 and runs the pipeline
- missing/wrong → responds **401** and stops

Laravel's `TriggerJobEmbeddingPipeline` already sends this header, so the real
job-publish flow is unaffected. **Manual test calls must now include the header**, e.g.:
```
-H "X-N8N-Webhook-Secret: talento-secret-2026"
```
(The internal Lane A → Lane C path bypasses the `/cv-match` webhook, so it's
unaffected by this gate.)

## Before it works end-to-end — verify these

- **Bearer token**: set to `1|…q8…` (the value WF1 used for the sanctum-guarded
  `/api/v1/resumes/*` routes). WF2/WF3 used `…k8…`; confirm which is the real
  Sanctum token. The `/api/n8n/*` routes ignore the bearer (secret-guarded).
- **Webhook secret** matches `N8N_WEBHOOK_SECRET=talento-secret-2026` in Laravel `.env`.
- **Qdrant collections** `talento_cv_embeddings` / `talento_job_embeddings` use
  **Cosine** distance, else scores can exceed 1 and `match/done` 422s
  (`match_score` is validated `between:0,1`).
- **`users.google_drive_file_id`** is populated for candidates — `matchDone`
  resolves Drive id → user and silently skips unknown ids.
- Qdrant `score_threshold: 0.55` vs Laravel `SCORE_SUGGEST=0.60`: matches in
  0.55–0.60 are stored as `new` but hidden by `visible()` / the controller's
  default `min_score`. Lower the threshold dependence or align the numbers if
  you want those surfaced.
