توثيق موديلات منصة TALENTO

هذا المستند يوثّق كل موديل Eloquent داخل المجلد app/Models في منصة TALENTO للتوظيف (Laravel). لكل موديل: الجدول المرتبط، الغرض، الحقول القابلة للتعبئة (Fillable)، الحقول المخفية (Hidden)، تحويلات الأنواع (Casts)، العلاقات بالتفصيل، النطاقات المحلية (Scopes)، الـ Accessors/Mutators، الأحداث (Events)، والـ Traits المستخدمة.

ملاحظات معمارية عامة

  • جميع الموديلات (عدا User و EmailLog) ترث من Illuminate\Database\Eloquent\Model الافتراضي.
  • User يرث من Illuminate\Foundation\Auth\User (Authenticatable) ويُطبّق MustVerifyEmail.
  • تُعرّف الـ Casts عبر دالة casts() المحمية (أسلوب Laravel 11/12) بدلاً من خاصية $casts.
  • لا يوجد أي موديل يحتوي على Accessors/Mutators صريحة أو دالة booted() لأحداث الموديل؛ منطق الأعمال يقع في طبقة الخدمات (Services).
  • الجداول الوسيطة: job_seeker_skill، job_post_skill، job_skill، course_skill (ولا توجد لها موديلات مستقلة).

User

extends Authenticatable implements MustVerifyEmail

app/Models/User.php — table: users

نظرة عامة

الموديل المركزي للمصادقة. يمثّل كل حساب في النظام بغضّ النظر عن دوره (admin، company_owner، company_member، job_seeker) عبر الحقل role. يرث من Illuminate\Foundation\Auth\User ويُطبّق واجهة MustVerifyEmail لإلزام التحقق من البريد. يوفّر دوال مساعدة للأدوار ويربط الملفات الفرعية لكل دور.

Traits المستخدمة

  • HasApiTokens — توكنات Sanctum للمصادقة عبر API.
  • HasFactory — دعم المصانع (Factories) للاختبارات/البذور.
  • Notifiable — إرسال الإشعارات للمستخدم.

Casts

  • email_verified_at → datetime
  • last_login_at → datetime
  • password → hashed
  • is_active → boolean

Fillable

email, phone, password, role, is_active, last_login_at, email_verified_at

Hidden

password, remember_token

الدوال المساعدة وأحداث الموديل

  • sendEmailVerificationNotification() — يُعيد تعريف الإرسال لاستخدام إشعار مخصّص ومُدرَج في الطابور: VerifyEmailNotification بدلاً من إشعار Laravel الافتراضي.
  • isAdmin(), isCompanyOwner(), isCompanyMember() (owner أو member), isJobSeeker(), hasRole($roles) — فحص الدور.

لا توجد دالة booted() أو Observers مسجّلة داخل الموديل.
